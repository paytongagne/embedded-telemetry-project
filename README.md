# Embedded Telemetry Project

This repository is being rebuilt into a portfolio project focused on Python, connected-device data, backend processing, and software documentation.

## Goal

Build a small system that creates sample device readings, processes the data, and prepares it for future database and dashboard features.

## Tech Stack

- Python
- JSON
- Backend data processing
- Testing
- Documentation

## Planned Features

- Simulated device readings
- Data validation
- JSON output
- Basic alert labels
- Unit tests
- Future database storage
- Future dashboard page

## Portfolio Purpose

This project is meant to show practical software development around embedded-style data, backend workflows, and clean project organization.
